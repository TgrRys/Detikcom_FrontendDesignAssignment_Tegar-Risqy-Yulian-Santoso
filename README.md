    ```bash

    ```bash
    cd Detikcom_FrontendDesignAssignment_Tegar-Risqy-Yulian-Santoso

3. **Instal dependensi**

    ```bash
    npm install

4. **Jalankan Tailwind CLI Build**

    npm run dev
